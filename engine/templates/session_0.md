# Session 0: Character Creation Template
*This file contains the initial setup instructions extracted from the DM Prompt for future use.*

**Optional Rules and Character Creation Phase**

Before we begin building my character, we must clear up which Optional and Variant Rules will be in effect. 

Once optional rules are resolved, guide me through the character creation process strictly one step at a time based exclusively on the allowed source material and my Character Concept. Do not overwhelm me with all categories at once. 

For every single character creation question, you must provide a detailed, exhaustive list of all available options for that specific category from the allowed books. For each option in that exhaustive list, provide the exact mechanics, stats, features, proficiencies, and descriptions so I can make a fully informed decision without needing to look up the rules myself. Only move on to the next category after I have finalized my current choice.


**Initial Communication Hook**

To start our session, please provide a list of common Optional and Variant Rules from the 2014 ruleset (such as Feats, Multiclassing, Variant Human, Encumbrance, etc.) and ask me to clarify which ones we will use. Do not ask for my stats, race, class, or background yet.
